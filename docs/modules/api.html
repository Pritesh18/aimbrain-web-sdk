Module api

API module

This module contains SDK credential management and type definitions used in the SDK. Please refer to each type definition for more detailed description.

SDK credentials

Method setKeyAndSecret is used to set API key and secret. This method must be invoked before using SDK methods resulting in API calls.

Example usage:

Aimbrain.api.setKeyAndSecret("[api key]", "[api secret]");

Setting credentials is not required if calls are made with request serialization.
